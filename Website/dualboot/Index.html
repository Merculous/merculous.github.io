Dualbooting is a process in which we take advantage of iBEC/iBoot's ota upgrade routines. This is because when you're upgrading your device via ota (over-the-air), iBEC\iBoot uses specific files and mount points which we can tinker with. And instead of upgrading the main os, we're telling iBEC/iBoot that we want the "upgraded" os at a specific partition, thus creating our secondary os.
